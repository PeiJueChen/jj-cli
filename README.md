# jj-cli
> A Tool CLI

## Install

```ruby
    npm i jj-cli -g
```

## Usage git 

```ruby
    j -h (help)
    j gpush <messgae> -b [branch] -d [depository]

    j gpull -b [branch] -d [depository]
```

## Usage open ios/platform
* for: cordova app / capacitor app

```ruby

    j open ios
    j open android
    
```
